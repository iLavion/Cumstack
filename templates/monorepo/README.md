# My cumstack App

A monorepo project with cumstack frontend and API backend.

## Structure

- `app/` - cumstack frontend application
- `api/` - API backend (to be implemented)

## Getting Started

1. Install dependencies:

   ```bash
   bun install
   ```

2. Start development server:

   ```bash
   bun run dev
   ```

3. Build for production:
   ```bash
   bun run build
   ```
